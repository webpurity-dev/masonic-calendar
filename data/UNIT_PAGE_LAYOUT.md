# Unit Page Layout Specification

## Page Structure
- Page Size: A6
- Margins: 20mm all sides
- Font: Helvetica or similar

## Header Section (120px height)
- Unit Name: Large bold title (24pt)
- Unit Number: Large bold title (24pt), after the Unit Name
- Alignment: Both Unit Name and Number center aligned

## Summary Section (100px height)
- Location Name: Bold (12pt)
- Address details: Regular (12pt)
- Contact details: Regular (12pt), unit email
- Alignment: All center aligned
